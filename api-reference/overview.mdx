## What is Keyforge?
Keyforge is a SaaS platform that provides a suite of tools for managing and monitoring your API keys. It allows you to create, revoke, and monitor API keys for your applications with ultra low latency.

## Authentication

All API endpoints are authenticated using Bearer tokens. You can obtain a token by creating an API key in the Keyforge dashboard. The token should be included in the `Authorization` header of your requests.
